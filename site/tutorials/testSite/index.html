Canvas Tutorial

In this tutorial, we look at how to create a canvas and display an image within it. After displaying the image, we look at how to flip an image.

//create the canvas
var canvas = $('#myCanvas');
var context = canvas.get(0).getContext("2d");

//create an image and draw it 400x300 at 0,0 on the canvas
var image = new Image();
image.src = "image.png"
$(image).load(function () {
context.drawImage(image, 0 , 0, 400, 300);
});

//translate image and flip it
context.translate(400,300);
context.rotate(0.7854*4);
